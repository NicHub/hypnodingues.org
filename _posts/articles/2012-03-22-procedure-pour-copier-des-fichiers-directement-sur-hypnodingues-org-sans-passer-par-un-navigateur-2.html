Cette procédure s’applique aussi bien pour les Mac que les PC.

  1. Téléchargez le logiciel Cyberduck.

    Lien direct pour Mac
    Lien direct pour PC

  2. Installez Cyberduck.
  3. Connectez-vous à http://hypnodingues.org/wp-login.php
  4. Téléchargez le fichier hypnodingues.duck
  5. Double-cliquez sur le fichier hypnodingues.duck
  6. Si Cyberduck vous demande de passer en mode sécurisé, refusez.
  7. Le mot de passe est hypno1234
  8. Glissez-déposez dans la fenêtre de Cyberduck, les fichiers que vous voulez téléverser.
  9. Vous pouvez créer un signet pour que Cyberduck se souvienne des informations de connection en allant dans le menu Signet/Nouveau Signet.
