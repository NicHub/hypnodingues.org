Cette procédure doit être appliquée une fois. Ensuite, il suffira de cliquer sur le raccourci dans la fenêtre de Windows Explorer. Je l'ai testée dans Windows 7.
  1. Ouvrir Windows Explorer
  2. Appuyer sur la touche Alt pour faire apparaître la barre de menu.
  3. Cliquer sur Outils/Connecter un lecteur réseau.
  4. Dans la fenêtre qui apparaît, cliquer sur "Se connecter à un site Web permettant de stocker des documents et des images."
  5. Cliquer sur Suivant (2x).
  6. Adresse réseau ou Internet : ftp://hypnodingues%40hypnodingues.org@hypnodingues.org/
  7. Entrez le nom pour cet emplacement réseau : hypnodingues.org
  8. Cliquer sur Suivant puis sur Terminer.
  9. Dans la partie de gauche des fenêtres de Windows Explorer, cliquer sur hypnodingues.org.

Nom d'utilisateur : hypnodingues@hypnodingues.org
Mot de passe : hypno1234
Cliquer sur Enregistrer le mot de passe (sauf si l'ordi est utilisé par d'autres personnes)

Une fois que le raccourci est présent, il suffit de cliquer dessus et de copier-coller les fichiers depuis votre ordinateur vers le site.
